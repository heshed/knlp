KNLP 데몬
==============

* http://konlpy.org 라이브러리를 이용한 한국어 형태소 분석기 데몬

Install
--------------

    clone knlp
    pip -r requirements.txt

Run
--------------

    twistd web —-wsgi=knlp.app —-logfile=knlp.log -p tcp:8888

Usage
--------------

* open http://localhost:8888?q=input_your_query
